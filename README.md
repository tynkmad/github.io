## üöÄ Getting Started

This portfolio is ready to be deployed on GitHub Pages. Simply push to the repository and enable GitHub Pages in the repository settings.

### Local Development

1. Clone the repository
2. Open `index.html` in your browser, or
3. Serve with a local web server:
   ```bash
   python3 -m http.server 8080
   ```
4. Visit `http://localhost:8080`

## ‚úèÔ∏è Customization

To personalize this portfolio with your information:

### Update Personal Information

1. **index.html** - Line 28-30: Update hero title and subtitle
2. **index.html** - Line 44-47: Update About Me section
3. **index.html** - Line 143: Update email address
4. **index.html** - Line 148: Update GitHub username (already set to `tynkmad`)
5. **index.html** - Line 151: Update LinkedIn profile URL

### Update Skills

Edit the skills lists in the Skills section (lines 54-84) with your actual technologies and expertise.

### Update Projects

Replace the placeholder projects (lines 90-131) with your actual projects:
- Project name and description
- Demo and repository links
- Technology tags

### Customize Colors

Edit `styles.css` CSS variables in the `:root` section (lines 6-13) to change the color scheme:
- `--primary-color`: Main theme color
- `--secondary-color`: Accent color
- Other color variables for text, backgrounds, etc.
